# JOURNAL schema

Filename: `.ai/JOURNAL.md`. Append-only history (never edit past entries). New entries are prepended at the top (newest first).

Entry format:

```markdown
## YYYY-MM-DD HH:MM — <stage> — <plan_id or n/a>
- Did: <what was done>
- Learned: <what was learned, if anything>
- Decided: <decisions, with rationale>
- Next: <handoff to next session>
```
